Generates a Builder pattern implementation for a Go struct.

Creates a builder interface with fluent With* setter methods for each selected field, a private implementation struct, a constructor, and a B() method that builds the final struct instance. Supports generics (type parameters).

Place the caret on a struct type name and invoke the intention. A dialog will appear allowing you to select which fields to include in the builder. Exported fields are unchecked by default, unexported fields are checked.

The generated code follows the idiomatic Go builder pattern with an exported interface and an unexported implementation, enabling compile-time enforcement of the builder contract.
